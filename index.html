💡 คำแนะนำรูปแบบที่รองรับ:
• ตัวคั่น: เคาะวรรค, จุด (.), เท่ากับ (=), ลูกศร (->)
• สัญลักษณ์วันลาเต็มวัน (8 ชม.): V, S, P หรือ P8
• สัญลักษณ์ลากิจบางส่วน: ควบเครื่องหมายบวก เช่น 6+P2 หรือ 4+P4
• วันเดียวกันมีได้สูงสุด 3 ช่วงเวลา คั่นด้วยจุลภาค (,) เช่น 8-16,20-24
• 🔒 กฎเหล็กเวรดึก: พิมพ์ 20-24 จะเข้าช่อง OT ทันที และ 0-8 จะเข้าช่อง Duty เสมอ
📅 3. ระบบลงตารางเวรด่วนแบบล็อกวัน (สไตล์ Google Calendar)
เลือกวันที่ที่ต้องการลงเวรนี้ (สามารถคลิกเลือกซ้ำหลายๆ วันได้เหมือนการเลือกวันทำงานในระบบปฏิทิน)
* ระบบจะนำช่วงเวลานี้ไปใส่ในวันที่คุณเลือกไว้ พร้อม **"แยกเวลาส่วนที่เกินออกจากเวลาดิวตี้เพื่อไปคิดเป็น OT ให้อัตโนมัติ"** ตามกฎเหล็กเวรดึก (20-24 = OT เสมอ, 0-8 = Duty เสมอ) รองรับสูงสุด 3 ช่วงเวลาต่อวัน คั่นด้วยจุลภาค (,)
💰 4. ตั้งค่าราคาและอัตราค่าตอบแทน (ปรับเปลี่ยนได้ตลอดเวลา)
🎯 5. ปรับเป้าหมายชั่วโมง Duty และรีคำนวณเงิน OT ส่วนเกิน
* หากชั่วโมง Duty รวมสะสมเกินเป้าหมายที่ตั้งไว้ ระบบจะหักชั่วโมงส่วนเกินออกจากยอด Duty โดยอัตโนมัติ แล้วนำส่วนต่างนั้นไปแปลงเป็นเงิน OT เพิ่มเติมให้ทันที (เว้นว่างหรือใส่ 0 = ปิดการทำงานนี้)
📤 6. คัดลอกสรุปส่งไลน์ / บันทึกและเรียกคืนข้อมูล
* Export จะบันทึกทั้งตารางเวร, โปรไฟล์, โหมด, เรทราคา และเป้าหมาย Duty ไว้ในไฟล์เดียว เก็บไว้ย้อนดูภายหลัง หรือย้ายไปเปิดต่อในเครื่องอื่นได้เลย (Import ไฟล์เดิมกลับเข้ามาจะเขียนทับข้อมูลในตารางปัจจุบันทั้งหมด)
🧠 7. จัดการคลังคำแนะนำอัตโนมัติ (เพิ่ม/ลบล่วงหน้าได้ ไม่ต้องรอพิมพ์ในตารางก่อน)
